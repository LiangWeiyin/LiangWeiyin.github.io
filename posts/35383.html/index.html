1003. 阶乘后面0的数量

题目描述

n的阶乘后面有多少个0？

6的阶乘 = 12345*6 = 720，720后面有1个0。

分析

0只能由2和5相乘产生，统计2和5的个数，又因为2的个数肯定比5多，所以统计它们所能分解出的5的个数就行了。
但是可能有些数可以分解出多个5，以125为例：
125中，能整除5的有125/5=25个数，分别是

在这25个数中，每5个数中就有一个能被5^2=25整除，也就是它们能分解出2个5，这5个数分别是：

在这5个数中，有一个能被5^3=125整除，能分解出3个5

所以5的个数是25+5+1=31
所以125的阶乘相乘后有31个0

1004. n^n的末位数字

题目描述

给出一个整数N，输出N^N（N的N次方）的十进制表示的末位数字。

分析

先看看这组图片


可以看到
2^n的末位中，循环的规律是[2, 4, 8, 6]
3^n的末位中，循环的规律是[3, 9, 7, 1]
4^n的末位中，循环的规律是[4, 6]
5^n的末位中，循环的规律是[5]
6^n的末位中，循环的规律是[6]
7^n的末位中，循环的规律是[7, 9, 3, 1]
8^n的末位中，循环的规律是[8, 4, 2, 6]
9^n的末位中，循环的规律是[9, 1]
周期有1, 2, 4三个，所以可以取4为共同的周期，用n % 4， 把n^n的规模缩减到n^4，需要注意的是，当余数为0时，代表的时n^4，而不是n^0。

1082. 与7无关的数

题目描述

一个正整数，如果它能被7整除，或者它的十进制表示法中某个位数上的数字为7，则称其为与7相关的数。求所有小于等于N的与7无关的正整数的平方和。

例如：N = 8，<= 8与7无关的数包括：1 2 3 4 5 6 8，平方和为：155。

分析

题目看起来用暴力很好做，但是直接暴力枚举的话，会超时，因为有很多判断是不断重复地。我们利用输入中最大的数 max，求小于等于max的与7无关的正整数的平方和 sum。但是，要把从1到max的每一个数对应的sum记录下来，这样我们就可以计算完所有输入的sum而不必浪费重复的时间。这就是所谓的打表。

1887. 颜色精简

题目描述

有一个颜色集合，里面有若干个颜色值（RBG系统表示）。现在有若干个颜色值需要映射到集合中的某个值。具体的映射方法是找到集合中和它的颜色距离最小的进行映射。在RGB颜色系统中，颜色用 (R,G,B) 三元组表示，R,G,B是0到255之间的整数。两个颜色之间的距离定义如下：假设给定两个颜色的三元组分别是 (Ra,Ga,Ba) 和 (Rb,Gb,Bb) ，那么他们之间的距离 $D=\sqrt{(Ra−Rb)^2+(Ga−Gb)^2+(Ba−Bb)^2}$。

分析

也是暴力题，输入有些麻烦。分为两个列表，二重循环分别计算距离D取最小值

1888. 可译编码

题目描述

一个符号集合的编码被称为可译编码的条件是，集合中没有哪一个符号的编码是另外一个符号的编码的前缀。

例如，现在有一个集合中含有四种符号 {X, Y, P, Q} 他们对应的编码是：
X:01, Y:10, P:0010, Q:0000

那么这样的编码是可译编码。如果是下面这种情况：
X:11, Y:00, P:011, Q:0010

就不是可译编码，因为Y的编码是Q的编码的前缀。

分析

暴力，利用python字符串的 startswith()即可判断是否为前缀，另外，python的列表不可直接用=进行复制(如b=a)，需要用list, 如b=list(a)。
